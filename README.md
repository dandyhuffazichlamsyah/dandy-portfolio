# Dandy's Portfolio Website

This is the source code for the professional portfolio website of **Dandy Huffaz Ichlamsyah**.

## Overview
- **Stack**: HTML5, Vanilla JavaScript, Tailwind CSS (via CDN).
- **Design**: Modern, clean, and responsive design with Dark Mode support.
- **Font**: Inter (Google Fonts).

## Project Structure
```
dandy-portfolio/
├── index.html              # Main HTML file
├── assets/
│   ├── css/
│   │   └── style.css       # Custom styles & animations
│   ├── js/
│   │   └── script.js       # Interactive logic
│   ├── img/
│   │   ├── profile.jpg     # (Required) Your profile photo
│   │   ├── projects/       # Project screenshots
│   │   └── icons/          # Icons (if any)
│   └── docs/
│       └── CV-Dandy.pdf    # (Required) Your CV file
└── README.md
```

## How to Customize
1. **Profile Photo**: Place your photo as `assets/img/profile.jpg`.
2. **CV**: Place your PDF CV as `assets/docs/CV-Dandy.pdf`.
3. **Project Images**: Add screenshots to `assets/img/projects/` and update the `src` in `index.html`.
4. **Content**: Edit `index.html` to update text, links, and contact info.
